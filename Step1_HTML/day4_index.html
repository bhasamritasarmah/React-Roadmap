HTML supports 140 standard colors.

A color in HTML can also be specified using an RGB value, using the formula -
rgb(red, blue, green)
where each of the color values can range from 0 to 255.
Therefore, there can be a total of 256 * 256 * 256 = 16777216 colors!

Shades of grey can be created by giving equal values to all the three colors.

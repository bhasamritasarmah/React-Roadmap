HTML Tables

HTML tables allow web developers to arrange data into rows and columns.
Each table cell is defined by a <td> and a </td> tag, where td stands for table data.
Each row starts with a <tr> and ends with a </tr> tag, where tr stands for table row.
Table header cells are defined by a <th> and a </th> tag, where th stands for table header.
A table can contain all sorts of HTML elements like -

  • text
  • images
  • lists
  • links
  • other tables, and so on.

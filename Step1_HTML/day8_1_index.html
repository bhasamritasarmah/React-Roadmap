Table Borders

If I set a background colour for each cell and give the border the colour of the background then it gives the impression of an invisible border. With the CSS property border-radius, the table cells get rounded borders. With the CSS property border-style, one can set the appearance of the border. Here, I have made the cell borders dotted.

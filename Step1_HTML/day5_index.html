HTML and CSS

Cascading Style Sheets (CSS) is used to format the layout of a webpage. The word cascading means that the style applied to a parent element will also be applied to all the child elements, unless different style has been applied to them.

CSS can be added to HTML documents in 3 ways -
Inline: By using style attribute inside HTML elements.
Internal: By using a <style> element in the <head> section.
External: By using a <link> element to link to an external CSS file.

In this paragraph, I am using inline CSS to change to color of this text to aquamarine, using the color property of CSS in the style attribute.
I am using internal CSS by adding the style element inside the <head> section. I have mentioned the background-color of the webpage to be lightgoldenrodyellow there.
I am using external CSS file by adding several styles in the day5_styles.css file available in the same folder, and linking that file here in the <head> section.

The CSS border property defines a border around an HTML element.
The CSS padding property defines a padding (space) between the text and the border.
The CSS margin property defines a margin space outside the border.

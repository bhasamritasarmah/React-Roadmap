Table Sizes

HTML tables can have different sizes for each row, column, or the entire table. The HTML style attribute can be used with CSS properties width and height to specify the size of a table, row, or column. Using a percentage as a size unit for a width means how wide this element would be compared to its parent element. In the case, the parent element of <table> is <body>. The width of table is marked as 100%. To set the size of a specific column, style attribute is added to the <th> or <td> element. To set the height of a specific row, style attribute is is added to a specific row. Table headers are defined with <th> elements.

Table Headers

By default table headers are bold and centered, but that can be changed with the CSS property text-align. A header can span two or more columns with the use of the HTML colspan attribute. The <caption> tag, inserted immediately after the <table> tag, is used to add caption to a table.

Image Maps

The HTML <map> tag defines an image map, which is an image with clickable areas. The areas are defined with one or more <area> tags.

Background Image

The background-repeat property with the value no-repeat, ensures that the background image will not repeat itself.
The background-size property with the value cover, ensures that the image covers the whole element.
The background-attachment property with the value fixed, ensures that the entire element always remains covered, no matter what the size of the window.

Pictures in HTML

The HTML <picture> element allows to display different pictures for different devices or screen sizes. Also, in the case where browsers may not recognise all types of picture formats, using <picture> tag, one can add images of all formats, so that the browser uses the first format it recognises.

Favicons in HTML

A favicon is a small image displayed to the left of the page title in the browser tab.

HTML Page Title

The title is shown in the browser's title bar and should describe the content and the meaning of the page. Title page is very important for Search Engine Optimisation (SEO). The <title> element -
defines a title in the browser toolbar
provides a title for the page when it is added to favourites
displays a title for the page in search engine results.

Table Padding and Spacing

HTML tables can adjust the padding inside the cells, and also the space between the cells. Cell padding is the space between the cell edges and the cell content. By default, the padding is set to 0.
Cell spacing is the spacing between each cell. By default, the space is set to 2px.

Table Rowspan and Colspan

HTML tables have cells that can span over multiple rows or columns. To make a cell span over multiple columns, use the colspan attribute. To make a cell span over multiple rows, use the rowspan attribute. The values of the colspan and the rowspan represents the number of columns and rows to span, respectively.
